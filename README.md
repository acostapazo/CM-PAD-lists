# CM-PAD-lists
Protocol lists used in "Learning to Learn Face-PAD: a lifelong learning approach"

## Replace ENV variables

Replace the following environment variables with its related database path:
* REPLAY-MOBILE_PATH
* OULU-NPU_PATH
* MSU-MFSD_PATH
* REPLAY-ATTACK_PATH
* SIW_PATH
* ROSE-YOUTU_PATH
* CASIA-FASD_PATH
* 3DMAD_PATH
* HKBU_PATH
* CSMAD_PATH
